# Commodity Price Tracker Dashboard

An interactive **dashboard web app** built with **Python and Streamlit** to track and analyze commodity prices. This app allows users to explore trends in major commodities including **Gold, Silver, Crude Oil, Natural Gas, and Copper** using moving averages, normalized comparisons (base 100), and percentage change analysis.

# Features
- Track current and historical commodity prices.  
- Analyze trends with moving averages.  
- Compare multiple commodities with normalized (base 100) scaling.  
- View percentage changes over selected periods.  
- Interactive sidebar for commodity selection, intervals, and time periods.

# Tech Stack
- Python 3  
- Streamlit   
- pandas   
- Plotly   
- yfinance
